# Solutecno Argentina Repo

The new WhatsApp bot project is available in:

- `junio-whatsapp-premium-bot/`

Main files:

- `junio-whatsapp-premium-bot/index.js`
- `junio-whatsapp-premium-bot/package.json`
- `junio-whatsapp-premium-bot/rules.json`
- `junio-whatsapp-premium-bot/README.md`

This project is designed to:

- Work with QR login.
- Respond only to new private messages.
- Ignore groups, statuses, channels and newsletters.
- Avoid loops and old history replays.
- Allow programmable replies through `rules.json` or a webhook.
